# 📚 Tiếng Nhật Cơ Bản

## 1. Giới thiệu

Tiếng Nhật (日本語 - にほんご) là ngôn ngữ chính thức của Nhật Bản. Hệ thống chữ viết tiếng Nhật bao gồm:

* Hiragana (ひらがな)
* Katakana (カタカナ)
* Kanji (漢字)

---

## 2. Hiragana

### Bảng nguyên âm

| Hiragana | Romaji | Nghĩa |
| -------- | ------ | ----- |
| あ        | a      | a     |
| い        | i      | i     |
| う        | u      | u     |
| え        | e      | e     |
| お        | o      | o     |

### Ví dụ

| Từ vựng | Cách đọc | Nghĩa     |
| ------- | -------- | --------- |
| あさ      | asa      | Buổi sáng |
| いぬ      | inu      | Con chó   |
| うみ      | umi      | Biển      |
| えき      | eki      | Nhà ga    |
| おちゃ     | ocha     | Trà       |

---

## 3. Katakana

Katakana thường được dùng để viết:

* Từ mượn nước ngoài
* Tên nước ngoài
* Thuật ngữ kỹ thuật

### Ví dụ

| Katakana | Romaji    | Nghĩa                 |
| -------- | --------- | --------------------- |
| コンピュータ   | konpyu-ta | Máy tính              |
| テレビ      | terebi    | Tivi                  |
| コーヒー     | ko-hi-    | Cà phê                |
| スマホ      | sumaho    | Điện thoại thông minh |

## 5. Mẫu Câu Giao Tiếp

### Chào hỏi

#### Xin chào

```jp
こんにちは。
```

**Konnichiwa.**

→ Xin chào.

#### Chào buổi sáng

```jp
おはようございます。
```

**Ohayou gozaimasu.**

→ Chào buổi sáng.

#### Chào buổi tối

```jp
こんばんは。
```

**Konbanwa.**

→ Chào buổi tối.

---

## 6. Ngữ Pháp N5

### Mẫu câu 1

```jp
N1 は N2 です。
```

Ý nghĩa: N1 là N2.

Ví dụ:

```jp
私は学生です。
```

**Watashi wa gakusei desu.**

→ Tôi là học sinh/sinh viên.

---

### Mẫu câu 2

```jp
N1 の N2
```

Ý nghĩa: N2 của N1.

Ví dụ:

```jp
日本の車
```

**Nihon no kuruma**

→ Xe của Nhật.

## 8. Hội Thoại Mẫu

### Giới thiệu bản thân

A:

```jp
はじめまして。
```

Rất hân hạnh được gặp bạn.

B:

```jp
はじめまして。
```

Rất hân hạnh được gặp bạn.

A:

```jp

よろしくお願いします。
```
